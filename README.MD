## Como rodar o projeto ?

Para rodar o ambiente de desenvolvimento:

```bash
yarn
# ou
npm i
# e
npm run dev
# ou
yarn dev
```

Abra [http://localhost:5173/](http://localhost:5173) com seu navegador para ver o resultado.
